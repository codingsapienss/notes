# Knowledge Base 📚

A personal knowledge hub for software engineering insights, articles, and technical handbooks. Built with [Docusaurus](https://docusaurus.io/).

### 🚀 Live Site
**[https://codingsapienss.github.io/notes/](https://codingsapienss.github.io/notes/)**

## 📂 Project Structure

- `docs/dsa`: Data Structures and Algorithms handbook (C++ focused).
- `docs/web-dev`: Web development insights covering React, TypeScript, and modern tools.
- `src/pages`: Custom landing page and static pages.
- `static/img`: Project assets and technology logos.

## 🛠️ Local Development

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run start
   ```

3. **Build the production site:**
   ```bash
   npm run build
   ```

## 🚢 Deployment

The project is automatically deployed to GitHub Pages using the `npm run deploy` command.
