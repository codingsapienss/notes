Counting Set Bits (Population Count)

Counting Set Bits is one of the most important topics in Bit Manipulation.

A Set Bit means:

Bit = 1

Example:

13

Binary:

1101

Number of set bits:

3

because:

1 1 0 1
↑ ↑   ↑

contains three 1s.

What is a Set Bit?​

A bit having value:

1

is called a:

Set Bit

What is an Unset Bit?​

A bit having value:

0

is called:

Unset Bit

or

Clear Bit

Example​

13

Binary:

1101

Count:

1 1 0 1

=
3 Set Bits

Method 1 — Division by 2 (Basic Approach)​

Idea​

Convert number into binary.

While converting:

If remainder = 1

Count++

Dry Run​

Example:

13
13 % 2 = 1

Count = 1
6 % 2 = 0

Count = 1
3 % 2 = 1

Count = 2
1 % 2 = 1

Count = 3

Result:

3

Code​

int countSetBits(int n)
{
    int count = 0;

    while(n > 0)
    {
        if(n % 2 == 1)
        {
            count++;
        }

        n /= 2;
    }

    return count;
}

Example​

cout << countSetBits(13);

Output​

3

Complexity​

Time​

O(log₂N)

Space​

O(1)

Method 2 — Right Shift Technique​

Much more natural for Bit Manipulation.

Idea​

Check:

Last Bit

using:

n & 1

Then:

n >>= 1

Why Does It Work?​

Last bit:

1

means:

Set Bit

Example

13

1101

Last bit:

1

Check:

13 & 1
1101

0001
----
0001

Result:

1

Dry Run​

13

1101

Iteration 1

1101

Last Bit = 1

Count = 1

Shift

0110

Iteration 2

0110

Last Bit = 0

Shift

0011

Iteration 3

0011

Last Bit = 1

Count = 2

Shift

0001

Iteration 4

0001

Last Bit = 1

Count = 3

Result

3

Code​

int countSetBits(int n)
{
    int count = 0;

    while(n > 0)
    {
        count += (n & 1);

        n >>= 1;
    }

    return count;
}

Complexity​

ComplexityValue
TimeO(log₂N)
SpaceO(1)

Method 3 — Brian Kernighan Algorithm​

One of the most famous interview algorithms.

Core Observation​

Formula:

n = n & (n - 1)

removes:

Rightmost Set Bit

Example​

12

1100
11

1011

AND

1100

1011
----
1000

Notice:

One Set Bit Removed

Idea​

Instead of checking every bit:

Remove one set bit at a time.

Dry Run​

Example:

13

1101

Iteration 1

1101

↓

1100

Count:

1

Iteration 2

1100

↓

1000

Count:

2

Iteration 3

1000

↓

0000

Count:

3

Stop.

Result:

3

Code​

int countSetBits(int n)
{
    int count = 0;

    while(n)
    {
        count++;

        n = n & (n - 1);
    }

    return count;
}

Complexity​

Time​

O(Number of Set Bits)

Not:

O(log N)

Example:

1000000000000000

contains:

1 set bit

Only:

1 iteration

needed.

Why Is This Better?​

For sparse numbers:

Very Few Set Bits

it is significantly faster.

Method 4 — STL Built-in Function​

C++ provides optimized CPU-level implementations.

For int​

__builtin_popcount(n)

Example

cout << __builtin_popcount(13);

Output:

3

For long long​

__builtin_popcountll(n)

Example

long long num = 10000000000LL;

cout << __builtin_popcountll(num);

Complexity​

Typically:

O(1)

or

Very Fast Hardware Optimized

depending on architecture.

Comparison of All Methods​

MethodTime ComplexitySpace
Division by 2O(log₂N)O(1)
Right ShiftO(log₂N)O(1)
Brian KernighanO(Set Bits)O(1)
STL PopcountOptimizedO(1)

Which Method Should You Use?​

Interview​

Use:

Brian Kernighan

because it demonstrates understanding.

Production Code​

Use:

__builtin_popcount()

when available.

Special Case — Count Set Bits from 1 to N​

Example:

1 → 5
1 = 1     → 1

2 = 10    → 1

3 = 11    → 2

4 = 100   → 1

5 = 101   → 2

Total:

7

This is a separate advanced problem often asked in interviews.

Common Mistakes​

Forgetting Parentheses​

Wrong

n & n - 1

Correct

n & (n - 1)

Infinite Loop​

Wrong

while(n)
{
    count++;
}

Missing:

n = n & (n - 1);

Using popcount for long long​

Wrong

__builtin_popcount(num);

when:

num = long long

Correct

__builtin_popcountll(num);

Interview Questions​

Q1​

How do you count set bits using Bit Manipulation?

Answer​

while(n)
{
    count++;

    n &= (n - 1);
}

Q2​

What does:

n & (n - 1)

do?

Answer​

Removes the rightmost set bit.

Q3​

Which counting set bits algorithm is most famous?

Answer​

Brian Kernighan Algorithm

Q4​

Time complexity of Brian Kernighan?

Answer​

O(Number of Set Bits)

Q5​

Best STL function?

Answer​

__builtin_popcount()

__builtin_popcountll()

Cheat Sheet​

// Method 1
n % 2
n /= 2
// Method 2
n & 1

n >>= 1
// Method 3
n &= (n - 1)
// STL
__builtin_popcount(n)
// STL long long
__builtin_popcountll(n)

Key Takeaways​

  • A Set Bit is a bit having value 1.
  • Counting set bits is one of the most common bit manipulation problems.
  • Division and Right Shift methods both work in O(log₂N).
  • Brian Kernighan's Algorithm works in O(Number of Set Bits).
  • n & (n-1) removes the rightmost set bit.
  • __builtin_popcount() is the fastest and simplest solution in C++.
  • Understanding set bit counting is essential for many advanced interview problems.
