Problem 1 — Minimum Bit Flips to Convert a Number

Problem Statement​

Given:

start = 10
goal  = 7

Find minimum bit flips required to convert:

10 → 7

Observation

A bit needs flipping only when:

Bits are different.

XOR Property

Same Bits      → 0

Different Bits → 1

Therefore:

start ^ goal

directly tells us:

Which bits differ.

Example

start = 10

1010
goal = 7

0111

XOR

1010

0111
----
1101

Count Set Bits

1101

contains:

3 set bits

Answer

3

Code

int minBitFlips(int start, int goal)
{
    int diff = start ^ goal;

    return __builtin_popcount(diff);
}

Complexity​

Time​

O(log(start ^ goal))

Space​

O(1)

Problem 2 — Power Set Using Bit Manipulation

Problem Statement​

Given:

[1,2,3]

Generate all subsets.

Observation

For:

n elements

there are:

2^n subsets

Binary Representation

For:

n = 3

Masks:

000
001
010
011
100
101
110
111

Each bit decides:

Include Element ?

1 → Yes

0 → No

Example

Mask:

101

Array:

[1,2,3]

Interpretation

1 → Include 1

0 → Skip 2

1 → Include 3

Subset:

[1,3]

Complete Dry Run

MaskSubset
000[]
001[1]
010[2]
011[1,2]
100[3]
101[1,3]
110[2,3]
111[1,2,3]

Code

vector<vector<int>> subsets(vector<int>& nums)
{
    int n = nums.size();

    vector<vector<int>> answer;

    for(int mask = 0; mask < (1 << n); mask++)
    {
        vector<int> subset;

        for(int i = 0; i < n; i++)
        {
            if(mask & (1 << i))
            {
                subset.push_back(nums[i]);
            }
        }

        answer.push_back(subset);
    }

    return answer;
}

Complexity

Time​

O(N × 2^N)

Space​

O(2^N)

Problem 3 — Single Number I

Problem Statement​

Every element appears:

Twice

except one element.

Find that element.

Example

[4,1,2,1,2]

Answer:

4

Brute Force

Use Hash Map.

Complexity​

O(N)

Space:

O(N)

Optimal XOR Solution

Key Property

A ^ A = 0

A ^ 0 = A

Example

4 ^ 1 ^ 2 ^ 1 ^ 2

Rearrange

4 ^ (1 ^ 1) ^ (2 ^ 2)
4 ^ 0 ^ 0
4

Code

int singleNumber(vector<int>& nums)
{
    int answer = 0;

    for(int num : nums)
    {
        answer ^= num;
    }

    return answer;
}

Complexity

Time​

O(N)

Space​

O(1)

Problem 4 — Single Number II

Problem Statement​

Every element appears:

3 times

except one element.

Find that unique element.

Example

[2,2,2,5]

Answer:

5

Brute Force

Hash Map.

Time:

O(N)

Space:

O(N)

Better Bit Counting Solution

Observation​

If every number appears:

3 times

then every bit count should be:

Multiple of 3

except bits belonging to the unique number.

Example

2 = 0010

2 = 0010

2 = 0010

5 = 0101

Bit Count

Position 0 → 1

Position 1 → 3

Position 2 → 1

Position 3 → 0

Take:

count % 3

Remaining bits form:

0101

which is:

5

Code

int singleNumber(vector<int>& nums)
{
    int answer = 0;

    for(int bit = 0; bit < 32; bit++)
    {
        int count = 0;

        for(int num : nums)
        {
            if(num & (1 << bit))
            {
                count++;
            }
        }

        if(count % 3)
        {
            answer |= (1 << bit);
        }
    }

    return answer;
}

Complexity

Time​

O(32 × N)

≈ O(N)

Space​

O(1)

Problem 5 — Single Number III

Problem Statement​

Every number appears:

Twice

except:

Two Numbers

which appear once.

Example

[1,2,1,3,2,5]

Answer:

[3,5]

Step 1

Take XOR of all numbers.

1 ^ 2 ^ 1 ^ 3 ^ 2 ^ 5

Pairs cancel.

Result:

3 ^ 5

Assume:

3 = 011

5 = 101
3 ^ 5

110

Step 2

Find Rightmost Set Bit

mask = xorValue & (-xorValue);

Example

110

Rightmost Set Bit:

010

Why?

Because:

3 and 5 differ here.

Step 3

Create Two Buckets

Group numbers by:

Set
or
Unset

at that bit.

Bucket 1

1
1
5

Bucket 2

2
2
3

Now XOR each bucket separately.

Pairs disappear.

Result:

3

5

Code

vector<int> singleNumber(vector<int>& nums)
{
    int xorAll = 0;

    for(int num : nums)
    {
        xorAll ^= num;
    }

    int mask = xorAll & (-xorAll);

    int a = 0;
    int b = 0;

    for(int num : nums)
    {
        if(num & mask)
        {
            a ^= num;
        }
        else
        {
            b ^= num;
        }
    }

    return {a, b};
}

Complexity

Time​

O(N)

Space​

O(1)

Common Interview Questions

Q1​

Why does XOR solve Single Number I?

Answer​

A ^ A = 0

A ^ 0 = A

Q2​

Why does Power Set contain 2^N subsets?

Answer​

Each element has:

2 choices

Take
Skip

Q3​

How do you isolate the rightmost set bit?

Answer​

n & (-n)

Q4​

How do you count differing bits between two numbers?

Answer​

start ^ goal

then count set bits.

Q5​

Which bit manipulation problem appears most frequently?

Answer​

Single Number

variants.

Key Takeaways

  • XOR is the most important operator for interview problems.
  • Power Set generation is based on bit masking.
  • Single Number I is solved using XOR cancellation.
  • Single Number II is solved using bit counting.
  • Single Number III is solved using XOR bucket partitioning.
  • Minimum Bit Flips uses XOR + Popcount.
  • Recognizing patterns is more important than memorizing solutions.
  • Most bit manipulation interview problems reduce to a few core identities.
