Part 6D — Common Bit Manipulation Techniques

This chapter contains the most important bit manipulation tricks asked in coding interviews.

Most of these operations can be performed in:

O(1)

time using Bitwise Operators.

These techniques form the foundation for:

  • Competitive Programming
  • DSA Interviews
  • Low-Level Programming
  • System Programming

Binary Position Refresher​

Consider:

13

Binary:

1101

Bit Positions:

3 2 1 0

1 1 0 1

Check if ith Bit is Set​

Problem​

Determine whether:

ith bit

is:

1 (SET)

or

0 (UNSET)

Method 1 — Left Shift Mask​

Formula​

n & (1 << i)

Logic​

Create a mask having:

Only ith bit = 1

Example:

n = 13

1101

Check:

i = 2

Step 1​

Create Mask

1 << 2

Binary:

0100

Step 2​

AND

1101

0100
----
0100

Result:

Non-Zero

Therefore:

Bit is SET

Code​

bool isSet(int n, int i)
{
    return (n & (1 << i)) != 0;
}

Example​

cout << isSet(13, 2);

Output:

1

Method 2 — Right Shift​

Formula​

(n >> i) & 1

Logic​

Bring the desired bit to the last position.

Example

13

1101

Check:

i = 2

Step 1​

Shift

1101 >> 2

0011

Step 2​

AND with 1

0011

0001
----
0001

Result:

1

Bit is set.

Code​

bool isSet(int n, int i)
{
    return ((n >> i) & 1);
}

Complexity​

ComplexityValue
TimeO(1)
SpaceO(1)

Set ith Bit​

Problem​

Make ith bit:

1

regardless of its current value.

Formula​

n | (1 << i)

Example​

n = 9

1001

Set:

i = 1

Mask:

0010

OR

1001

0010
----
1011

Result

11

Code​

int setBit(int n, int i)
{
    return n | (1 << i);
}

Clear ith Bit​

Problem​

Make ith bit:

0

regardless of current value.

Formula​

n & ~(1 << i)

Example​

n = 13

1101

Clear:

i = 2

Mask

1 << 2

0100

Invert

1011

AND

1101

1011
----
1001

Result

9

Code​

int clearBit(int n, int i)
{
    return n & ~(1 << i);
}

Toggle ith Bit​

Problem​

Flip the bit.

Meaning:

0 → 1

1 → 0

Formula​

n ^ (1 << i)

Example​

13

1101

Toggle:

i = 2

Mask:

0100

XOR

1101

0100
----
1001

Result

9

Code​

int toggleBit(int n, int i)
{
    return n ^ (1 << i);
}

Remove Rightmost Set Bit​

One of the most famous tricks.

Formula​

n & (n - 1)

Example​

n = 12

1100

Step 1​

n-1

1011

Step 2​

AND

1100

1011
----
1000

Result

8

Why Does This Work?​

Observation:

n

xxxxx1000
n-1

xxxxx0111

The first set bit gets removed.

Code​

int removeLastSetBit(int n)
{
    return n & (n - 1);
}

Check if Number is Power of 2​

Observation​

Power of 2 always contains:

Exactly One Set Bit

Examples

1  = 0001

2  = 0010

4  = 0100

8  = 1000

Notice:

n & (n-1)

always becomes 0

Formula​

n > 0 &&
(n & (n - 1)) == 0

Example​

8

1000
7

0111

AND

1000

0111
----
0000

Result

Power of 2

Code​

bool isPowerOfTwo(int n)
{
    return n > 0 &&
           (n & (n - 1)) == 0;
}

Swap Two Numbers Without Extra Variable​

XOR Method​

Initial

a = 5

b = 7

Step 1​

a = a ^ b;

Step 2​

b = a ^ b;

Becomes

(a ^ b) ^ b

= a

Step 3​

a = a ^ b;

Becomes

(a ^ b) ^ a

= b

Code​

int a = 5;
int b = 7;

a ^= b;
b ^= a;
a ^= b;

Result

a = 7

b = 5

Why XOR Swap is Rarely Used Today​

Modern approach:

swap(a, b);

is:

  • More readable
  • Compiler optimized
  • Safer

Extract Rightmost Set Bit​

Formula​

n & (-n)

Example​

12

1100

Two's Complement:

0100

AND

1100

0100
----
0100

Result

4

Meaning:

Rightmost Set Bit

was isolated.

Common Interview Questions​

Check Odd or Even​

n & 1

If:

1

Odd.

If:

0

Even.

Multiply by 2​

n << 1

Divide by 2​

n >> 1

Multiply by 8​

n << 3

Divide by 8​

n >> 3

Summary Table​

| Operation | Formula | | -------------------- | -------------- | ------- | | Check ith Bit | n & (1<<i) | | Check ith Bit | (n>>i)&1 | | Set ith Bit | n | (1<<i) | | Clear ith Bit | n & ~(1<<i) | | Toggle ith Bit | n ^ (1<<i) | | Remove Last Set Bit | n & (n-1) | | Extract Last Set Bit | n & (-n) | | Power of 2 | (n&(n-1))==0 | | Odd Check | n & 1 |

Interview Questions​

Q1​

How do you check whether the 5th bit is set?

n & (1 << 5)

Q2​

How do you remove the rightmost set bit?

n & (n - 1)

Q3​

How do you check if a number is a power of 2?

(n & (n - 1)) == 0

Q4​

How do you toggle a bit?

n ^ (1 << i)

Q5​

How do you set a bit?

n | (1 << i)

Cheat Sheet​

// Check Bit
n & (1 << i)
// Set Bit
n | (1 << i)
// Clear Bit
n & ~(1 << i)
// Toggle Bit
n ^ (1 << i)
// Remove Last Set Bit
n & (n - 1)
// Extract Last Set Bit
n & (-n)
// Power Of Two
(n & (n - 1)) == 0

Key Takeaways​

  • Most common bit tricks are based on AND, OR, XOR and shifts.
  • Checking, setting, clearing and toggling bits can all be done in O(1).
  • n & (n-1) is one of the most important formulas in bit manipulation.
  • A power of two contains exactly one set bit.
  • XOR has powerful cancellation properties.
  • Bit manipulation often converts seemingly complex problems into constant-time operations.
  • These formulas are among the most frequently asked in coding interviews.
