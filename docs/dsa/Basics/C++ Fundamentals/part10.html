Functions

Functions are one of the most important concepts in programming.

They help us:

  • Reuse code
  • Reduce duplication
  • Improve readability
  • Break large problems into smaller parts
  • Write modular programs

What is a Function?​

A Function is a reusable block of code that performs a specific task.

Without Functions​

cout << "Welcome" << endl;
cout << "Welcome" << endl;
cout << "Welcome" << endl;

Repeated code.

With Functions​

void printWelcome()
{
    cout << "Welcome" << endl;
}
printWelcome();
printWelcome();
printWelcome();

Real Life Analogy​

Think of a function like a machine.

Input
  ↓
Function
  ↓
Output

Example:

5 , 10
 ↓
Add Function
 ↓
15

Syntax of a Function​

returnType functionName(parameters)
{
    // code
}

Example​

void greet()
{
    cout << "Hello";
}

Function Declaration​

Also called:

Function Prototype

Tells the compiler that a function exists.

Example​

int add(int, int);

Function Definition​

Actual implementation.

Example​

int add(int a, int b)
{
    return a + b;
}

Function Call​

Invoking a function.

Example​

add(10, 20);

Complete Example​

#include <iostream>
using namespace std;

void greet()
{
    cout << "Hello World";
}

int main()
{
    greet();

    return 0;
}
Output​
Hello World

Function Execution Flow​

main()
   ↓
greet()
   ↓
Execute Function Body
   ↓
Return to main()

Return Type​

Defines what a function returns.

Void Function​

Returns nothing.

Example​

void greet()
{
    cout << "Hello";
}

Function Returning Value​

int getNumber()
{
    return 10;
}

Example​

int num = getNumber();

cout << num;
Output​
10

Parameters and Arguments​

Many beginners confuse these.

Parameters​

Variables defined in function definition.

Example​

void print(int num)
{
    cout << num;
}

Here:

num

is a parameter.

Arguments​

Actual values passed during function call.

Example​

print(50);

Here:

50

is an argument.

Example​

void greet(string name)
{
    cout << "Hello " << name;
}

int main()
{
    greet("Prashant");
}
Output​
Hello Prashant

Multiple Parameters​

int add(int a, int b)
{
    return a + b;
}

Example​

cout << add(10, 20);
Output​
30

Pass By Value​

Default behavior in C++.

Example​

void update(int num)
{
    num++;
}
int main()
{
    int value = 10;

    update(value);

    cout << value;
}
Output​
10

Why?​

Because:

A copy is created.

Memory:

value = 10

      Copy

num = 10

Different memory locations.

Pass By Reference (Introduction)​

Allows function to work on the original variable.

Example​

void update(int& num)
{
    num++;
}
int value = 10;

update(value);

cout << value;
Output​
11

Detailed discussion of references will be covered in later chapters.

Function Overloading​

Multiple functions with the same name but different parameters.

Example​

int add(int a, int b)
{
    return a + b;
}
double add(double a, double b)
{
    return a + b;
}

Usage​

cout << add(10, 20);

Output:

30
cout << add(2.5, 3.5);

Output:

6

Local Variables​

Variables declared inside a function.

Example​

void fun()
{
    int x = 10;
}

Scope​

x

can only be accessed inside:

fun()

Scope Example​

void fun()
{
    int x = 10;
}

int main()
{
    cout << x;
}

Compilation Error.

Call Stack (Basic Understanding)​

Whenever a function is called:

Memory is allocated
for that function.

Example:

main()
  ↓
add()

Stack:

┌───────┐
│ add() │
├───────┤
│ main()│
└───────┘

When:

add()

finishes,

its memory is removed.

Common Beginner Mistakes​

Forgetting Function Call​

void greet()
{
    cout << "Hello";
}

Nothing prints until:

greet();

is called.

Missing Return Value​

Wrong

int getNum()
{
}

Correct

int getNum()
{
    return 10;
}

Wrong Number of Arguments​

add(10);

when function expects:

add(int,int);

Compilation Error.

Interview Questions​

Q1. What is a function?​

Answer​

A reusable block of code designed to perform a specific task.

Q2. Difference between parameter and argument?​

Answer​

Parameters are variables in function definition.

Arguments are actual values passed during function call.

Q3. What is a function prototype?​

Answer​

A declaration that informs the compiler about a function before its use.

Q4. What is function overloading?​

Answer​

Multiple functions with the same name but different parameter lists.

Q5. What is pass by value?​

Answer​

A copy of the variable is passed to the function.

Key Takeaways​

  • Functions improve code reusability and modularity.
  • Every function has a return type, name, parameters, and body.
  • Function declaration informs the compiler about a function.
  • Function definition contains the actual implementation.
  • Function calls execute the function.
  • Parameters receive values from arguments.
  • Pass by value creates a copy.
  • Pass by reference works on the original variable.
  • Function overloading allows multiple functions with the same name.
  • Functions use the call stack during execution.
