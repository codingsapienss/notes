C++ Basics & Program Structure

Before learning data types, memory, arrays, pointers, and OOP, it is important to understand how a C++ program is structured and how execution begins.

Learning Objectives​

In this chapter, you will learn:

  • Structure of a C++ program
  • Header files
  • Preprocessor directives
  • Namespace
  • main() function
  • Statements
  • Comments
  • Keywords
  • Identifiers
  • Variables (Introduction)
  • Constants (Introduction)
  • Program execution flow

Your First C++ Program​

#include <iostream>

using namespace std;

int main()
{
    cout << "Hello World";

    return 0;
}
Output​
Hello World

Understanding Program Structure​

Every C++ program follows a basic structure:

#include <iostream>

using namespace std;

int main()
{
    // Program Code

    return 0;
}

Structure Diagram​

Header Files
      ↓
Namespaces
      ↓
main()
      ↓
Statements
      ↓
return

Header Files​

What Are Header Files?​

Header files contain declarations of functions, classes, and objects that can be used in a program.

Example​

#include <iostream>

Why Do We Need It?​

Without:

#include <iostream>

the compiler does not know about:

cout
cin
endl

Example:

cout << "Hello";

will produce a compilation error if <iostream> is not included.

Common Header Files​

Header FilePurpose
<iostream>Input and Output
<string>String Support
<vector>Dynamic Arrays
<cmath>Mathematical Functions
<iomanip>Output Formatting

Preprocessor Directives​

What Are They?​

Lines beginning with:

#

are called preprocessor directives.

They are processed before compilation.

Example​

#include <iostream>

Other examples:

#define PI 3.14
#ifdef
#ifndef

Important Point​

The compiler does not process preprocessor directives directly.

The:

Preprocessor

handles them before compilation starts.

Namespace​

What Problem Does It Solve?​

Different libraries may contain functions or objects with the same name.

Namespaces prevent naming conflicts.

Example​

using namespace std;

Now we can write:

cout << "Hello";

instead of:

std::cout << "Hello";

Without Namespace​

#include <iostream>

int main()
{
    std::cout << "Hello";
}

With Namespace​

#include <iostream>

using namespace std;

int main()
{
    cout << "Hello";
}

The main() Function​

What Is It?​

The entry point of every C++ program.

Program execution always starts from:

main()

Syntax​

int main()
{
}

Example​

int main()
{
    cout << "Program Started";

    return 0;
}
Output​
Program Started

Why Does main() Return int?​

int main()

means:

main returns an integer value.

Usually:

return 0;

indicates:

Program executed successfully.

return 0​

Example:

int main()
{
    cout << "Hello";

    return 0;
}
Meaning​
Return control to Operating System
with success status.

Statements​

What Is A Statement?​

A statement is an instruction given to the computer.

Example:

cout << "Hello";

Example:

int age = 20;

Most statements end with:

;

called:

Semicolon

Semicolon (;)​

Why Is It Needed?​

Marks the end of a statement.

Correct:

int age = 20;

Wrong:

int age = 20

Compilation Error.

Comments​

Comments are ignored by the compiler.

Used to explain code.

Single Line Comment​

// This is a comment

Example:

// Store user's age
int age = 20;

Multi-line Comment​

/*
This is
a multi-line
comment
*/

Example:

/*
Program:
Calculate Sum
Author:
Prashant
*/

Keywords​

What Are Keywords?​

Reserved words that already have a meaning in C++.

Examples:

int
double
char
return
if
else
for
while
class
void

Invalid Example​

int int = 10;

Compilation Error.

Because:

int

is a keyword.

Identifiers​

What Are Identifiers?​

Names given to:

  • Variables
  • Functions
  • Classes
  • Objects

Example:

int age = 20;

Identifier:

age

Identifier Rules​

Rule 1​

Can contain:

Letters
Digits
Underscores

Valid:

studentAge
_age
count123

Rule 2​

Cannot start with a digit.

Invalid:

123count

Valid:

count123

Rule 3​

Cannot be a keyword.

Invalid:

int return = 10;

Rule 4​

C++ is case-sensitive.

age
Age
AGE

are different identifiers.

Variables (Introduction)​

What Is A Variable?​

A named memory location used to store data.

Example:

int age = 20;

Here:

age

is the variable.

Value stored:

20

Variable Declaration​

int age;

Variable Initialization​

int age = 20;

Multiple Variables​

int a = 10;
int b = 20;
int c = 30;

Constants (Introduction)​

What Is A Constant?​

A value that cannot be changed after initialization.

Example

const int MAX_SIZE = 100;

Invalid:

MAX_SIZE = 200;

Compilation Error.

constexpr​

Modern C++ provides:

constexpr

Example:

constexpr double PI = 3.14159;

Used for compile-time constants.

Program Execution Flow​

Consider:

#include <iostream>

using namespace std;

int main()
{
    cout << "A" << endl;

    cout << "B" << endl;

    cout << "C" << endl;

    return 0;
}
Output​
A
B
C
Execution Flow​
Program Starts
      ↓
main()
      ↓
Statement 1
      ↓
Statement 2
      ↓
Statement 3
      ↓
return 0
      ↓
Program Ends

Common Beginner Mistakes​

Missing Semicolon​

Wrong:

int age = 20

Correct:

int age = 20;

Missing Header File​

Wrong:

cout << "Hello";

without:

#include <iostream>

Using Keyword As Variable Name​

Wrong:

int return = 10;

Forgetting main()​

Every C++ program must have:

main()

Interview Questions​

Q1. What is the entry point of a C++ program?​

Answer​
main()

Q2. Why do we use #include <iostream>?​

Answer​

To use input/output objects such as:

cin
cout
endl

Q3. What is a namespace?​

Answer​

A mechanism used to avoid naming conflicts.

Q4. What does return 0 mean?​

Answer​

Program executed successfully.

Q5. What is an identifier?​

Answer​

A user-defined name given to variables, functions, classes, etc.

Cheat Sheet​

#include <iostream>

using namespace std;

int main()
{
    cout << "Hello World";

    return 0;
}
// Single Line Comment

/*
Multi-line
Comment
*/
int age = 20;

const int MAX_SIZE = 100;

constexpr double PI = 3.14159;

Key Takeaways​

  • Every C++ program starts execution from main().
  • Header files provide declarations for library features.
  • #include is a preprocessor directive.
  • Namespaces help avoid naming conflicts.
  • Statements usually end with a semicolon (;).
  • Comments are ignored by the compiler.
  • Keywords are reserved words in C++.
  • Identifiers are names given to program entities.
  • Variables store data in memory.
  • Constants store values that cannot be modified.
  • Program execution follows a top-to-bottom flow inside main().
